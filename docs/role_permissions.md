# Dawrlk - Roles & Permissions (MVP)

## Roles
- customer
- owner
- admin

## Customer Permissions
- Search services
- View nearby businesses
- View business services (price/duration)
- View availability
- Create booking
- View own bookings
- Cancel own booking (policy later)

## Owner Permissions
- Create/manage business profile (only owned businesses)
- Add/update business services (price/duration)
- Set availability windows (days/hours)
- View bookings for owned business
- Cancel/mark booking status (policy later)

## Admin Permissions
- Manage all businesses
- Approve/suspend businesses (optional in MVP)
- Manage all services catalog
